# Simpsons App

Este proyecto es una app interactiva donde puedes explorar los personajes de la serie "Los Simpsons". Desarrollado con:

- **Vite** (como bundler)
- **Tailwind CSS** (para el estilo)
- **TypeScript** (para la lógica)
